# CSC_161_Personal_Project

## Description
This project is a **Video Game Version Review System**, built with **Django**.  
Users can create accounts, log in via custom authentication, and submit reviews for different versions of video games.  
The project also includes a dashboard, and search functionality.

## Features
- User registration and login (custom login)
- Create, view, and search video game version specific reviews
- PostgreSQL database integration via Docker
